# SEO Checklist — thehippiescientist.net

Run this checklist weekly until all items are resolved.

## Weekly

- [ ] Check GSC for new impressions/clicks (target: >0 clicks)
- [ ] Check Ahrefs for new backlinks (flag any new spam domains)
- [ ] Verify sitemap includes all 54 indexable entities
- [ ] Run lighthouse audit (target: >90 performance)

## Bi-Weekly

- [ ] Check crawl stats in Ahrefs Site Audit
- [ ] Review top 10 GSC keywords for position changes
- [ ] Check for new 404 errors
- [ ] Verify robots.txt and _redirects are working

## Monthly

- [ ] Full Ahrefs Site Audit crawl
- [ ] Backlink audit — add new spam domains to disavow
- [ ] Review indexability of all 54 entities
- [ ] Competitor backlink gap analysis
- [ ] Content freshness review — update pages older than 6 months

## Quarterly

- [ ] Full competitive analysis
- [ ] Expand sitemap allowlist with proven content
- [ ] Link building campaign — target 5 new legitimate backlinks
- [ ] Review and update disavow file
