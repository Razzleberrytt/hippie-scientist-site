# SEO Keyword Execution Plan — First Revenue Pages

Purpose: convert the long-tail keyword spreadsheet into a concrete publishing queue for The Hippie Scientist.

Strategic position: build purchase-adjacent evidence pages that generic health sites do not answer well. Prioritize comparison pages and skeptical `does it work` pages because they create clearer decisions than single-ingredient monographs.

Canonical host: `https://thehippiescientist.net`

## Non-negotiable rules

1. Do not invent medical certainty.
2. Do not claim herbs or supplements treat, cure, or replace medical care.
3. Keep affiliate CTAs secondary to evidence and safety.
4. Prefer human evidence, meta-analyses, RCTs, NIH/NCCIH/ODS/FDA references, and PubMed-linked citations.
5. Every page must include: decision summary, who it may fit, who should avoid it, safety notes, evidence strength, internal links, and FAQ schema.
6. Avoid aggressive supplement-bro language. The site voice should be calm, skeptical, and useful.

## Page 1 — Magnesium Glycinate vs L-Threonate for Sleep

- **Priority:** 1
- **Target keyword:** `magnesium glycinate vs magnesium l-threonate for sleep`
- **Recommended URL:** `/compare/magnesium-glycinate-vs-l-threonate-for-sleep`
- **Page type:** Comparison
- **Intent:** Commercial investigation + decision support
- **Primary affiliate angle:** magnesium glycinate, magnesium L-threonate / Magtein-style products, third-party-tested magnesium
- **Risk level:** Low to moderate; avoid claiming sleep disorder treatment.

### H1
Magnesium Glycinate vs L-Threonate for Sleep: Which One Makes More Sense?

### Meta title
Magnesium Glycinate vs L-Threonate for Sleep | The Hippie Scientist

### Meta description
A skeptical, evidence-based comparison of magnesium glycinate and magnesium L-threonate for sleep, stress, cognition, safety, and supplement selection.

### Decision summary
Magnesium glycinate is the more practical first choice for general sleep support, especially when the goal is relaxation, affordability, and tolerability. Magnesium L-threonate is more interesting when the sleep issue overlaps with cognition, mental fatigue, or brain-health positioning, but it is usually more expensive and should not be presented as automatically superior for sleep.

### Suggested above-the-fold table
| Factor | Magnesium glycinate | Magnesium L-threonate |
|---|---|---|
| Best fit | General relaxation and sleep routine | Sleep plus cognition/brain-health interest |
| Cost | Usually lower | Usually higher |
| Evidence fit | Practical, common sleep-support choice | More cognition-positioned; sleep claims need caution |
| Main caution | GI effects, medication interactions, kidney disease caution | Cost, overhyped brain claims, kidney disease caution |
| Bottom line | Better first pick for most people | Niche option if cognition angle matters |

### Required sections
1. Fast answer
2. What each form is
3. Evidence for sleep
4. Evidence for stress/calm
5. Evidence for cognition
6. Safety and who should avoid magnesium supplements
7. Which one should you choose?
8. What to look for in a product
9. FAQ

Does Lion’s Mane Actually Work for Brain Fog?

Does Lion’s Mane Work for Brain Fog? Evidence, Limits, and Buyer Notes

### Meta description
A skeptical review of lion’s mane mushroom for brain fog, focus, cognition, mechanisms, safety, and what to look for in a supplement.

### Decision summary
Lion’s mane is plausible for cognition support, but the brain-fog claim is usually stronger in marketing than in direct clinical evidence. The best page angle is not “yes it works”; it is “possibly useful, but quality and expectations matter.” This skeptical framing should convert because it matches searcher doubt.

### Required sections

- **Best fit:** mild focus/clarity support, not acute stimulant-like performance.
- **Not a fit:** severe cognitive symptoms, neurological red flags, or expecting immediate effects.
- **Evidence rating:** limited to moderate depending on endpoint; direct brain-fog evidence should be treated as limited.

## Implementation checklist for each page

For every page, implement the same structure:

1. Create or update the route.
2. Add metadata: title, description, canonical URL, open graph title/description.
3. Add the page to `app/sitemap.ts` if sitemap is manually enumerated.
4. Add internal links from relevant herb, compound, goal, compare, or guide pages.
5. Add a short evidence table with conservative language.
6. Add FAQ schema only for FAQs that appear visibly on the page.
7. Add affiliate CTA placeholders only after the evidence/safety sections.
8. Run:
   - `npm run validate:route-seo`
   - `npm run check`
   - `npm run build`

## Recommended build order

1. Build Page 1 first because it is low-risk and monetizable.
2. Build Page 2 second because it strongly matches the site’s evidence-driven compare engine.
3. Build Page 3 third because skeptical framing is a differentiator.
4. Build Page 5 fourth because it can internally link to Page 2.
5. Build Page 4 last because it is medically sensitive and needs the strictest review.

## Business note

Do not publish all 20 keyword pages as thin pages. Publish the first five as strong decision pages. If impressions begin but clicks are weak, revise titles and meta descriptions. If clicks occur but no affiliate clicks happen, improve product-selection boxes. If neither impressions nor clicks occur after indexing, the keyword assumptions are probably too optimistic and the next step should be Google Search Console-driven iteration, not more blind content production.
