# Mental-health SEO checklist

- [ ] Unique 30–65 character SEO title
- [ ] Unique 110–190 character meta description
- [ ] Canonical lowercase hyphenated slug
- [ ] Index/follow robots with large image and unrestricted snippet previews
- [ ] Canonical, Open Graph, Twitter, author, publisher, and category metadata
- [ ] Article, FAQ, Breadcrumb, CollectionPage, and ItemList schema where applicable
- [ ] Concrete 1200×630 structured-data image
- [ ] One descriptive H1 and logical H2/H3 hierarchy
- [ ] On-page contents navigation for long guides
- [ ] Related internal links and hub inclusion
- [ ] Citation-backed claims and visible references
