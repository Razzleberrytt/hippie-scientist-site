## 3. Content Freeze Plan

### What Is "Complete" — Stop Adding

These categories have sufficient depth. Additional entries add long-tail pages with thin search volume and dilute topical authority:

| Category | Status | Rationale |
|----------|--------|-----------|
| **Obscure psychoactive botanicals** (Blue Lotus, Calea, Mugwort, Amanita) | Freeze | Good for brand differentiation but no search volume; adding more doesn't grow the audience |
| **Rare Ayurvedic herbs** with no English-language search volume | Freeze | 290 herb profiles already; marginal herb #291 delivers near-zero incremental traffic |
| **Individual compound forms** (e.g., 15 magnesium chelates) | Freeze | Cover the top 3-4 forms per compound; additional forms fragment content without improving ranking |
| **Isolated alkaloids with no human evidence** | Freeze | C/D-tier evidence; creates liability without value |

### What to Stop Adding Immediately
- New herb profiles for herbs already covered by close relatives (e.g., more Panax ginseng subspecies when American Ginseng and Korean Ginseng are already covered)
- Blog posts in categories with no topical cluster (random "cultivar notes" that don't ladder up to a goal cluster)
- Education pages on mechanisms not linked to any current cluster hub

### Focus Areas Going Forward

**1. Depth over breadth in top 5 clusters**
Sleep, Anxiety/Stress, Cognition, Energy, and Inflammation are where the search volume is. Every new page should ladder into one of these five clusters. If it doesn't, ask why before building it.

**2. Comparison pages over new monographs**
The 907 herb/compound profiles are a moat. The next traffic lever is comparison pages — they capture high-intent, decision-stage queries that monographs don't. Target: 30+ comparisons total (currently 23).

**3. Hub blog posts for each cluster**
Each of the 7 clusters above needs one long-form, evidence-graded "state of the science" hub article that:
- Ranks for the broadest cluster keyword (e.g., "best supplements for anxiety")
- Links to all spoke pages (goal guide + top 5 compounds + 2-3 comparisons)
- Is updated annually with new study data

Priority order: Sleep → Anxiety/Stress → Cognition → Energy → Inflammation

**4. Evidence layer on existing content**
Before adding new pages, upgrade existing content with the Evidence Score badges and "Show Me The Studies" panels (see DIFFERENTIATION_STRATEGY.md). A page that shows its work outranks a page that doesn't, all else equal. This is high-ROI work on the existing 907 pages.

**5. One new cluster to open: Longevity/NAD+**
This is the fastest-growing supplement category (NMN, NR, resveratrol, CoQ10). The site has the compound profiles but no cluster hub. A "NAD+ and longevity supplements explained" hub page + longevity stack would open a new traffic channel.

## 4. Editorial Standards for New Content

Any new page built must meet all of these before publishing:

- [ ] Assigned to a cluster hub (Sleep / Anxiety / Cognition / Energy / Inflammation / Metabolic / Longevity)
- [ ] Evidence tier scored (A/B/C/D) and displayed
- [ ] At least 3 citations with PMID (or marked D-tier if traditional only)
- [ ] Goal context declared: what problem does this compound solve?
- [ ] Comparison candidates identified: what should this page link to?
- [ ] Internal links: links to cluster hub + at least 2 related compound pages
- [ ] No unqualified health claims (use evidence tier language: "evidence suggests", "one RCT found")
