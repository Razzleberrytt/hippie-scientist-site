## Content Production Workflow

### Week-by-Week Template

**Monday: Keyword Research & Briefing**
- Finalize 5 topic keywords for the week
- Research 3+ sources per topic
- Create brief with:
  - Target keyword
  - Search intent
  - Related queries
  - Content angle
  - Internal link opportunities

**Tuesday-Thursday: Content Creation**
- Write 2-3 pages per day (1,500-2,500 words each)
- Follow template exactly
- Include 3+ primary sources
- Add 3-5 internal links per page
- Optimize for featured snippets (comparison tables, quick answers)

**Friday: Review & Publishing**
- Edit for brand voice
- Validate internal links
- Check SEO basics
  - Meta description (155-160 characters)
  - H1 (once, target keyword)
  - Proper heading hierarchy
  - 2+ internal links
  - Image alt text
- Publish and submit to GSC

**Weekly Output: 10-15 pages**

## Content Template Standards

### Condition Page Template (2,000 words)

```

## What is [Condition]?
- Definition (100 words)
- Prevalence statistics
- Key symptom overview

## Why [Condition] Happens
- Biological mechanisms (500 words)
- Risk factors
- Lifestyle contributors

## How [Relevant System] Affects [Condition]
- System overview
- Connection to condition
- Evidence from [Source 1]

## Herbs for [Condition]
- Top 5-7 herbs
- How each helps (mechanism)
- Evidence summary per herb
- Links to individual herb pages

## Compounds that Help with [Condition]
- Top 3-5 compounds
- How each works
- Evidence per compound
- Links to compound pages

## [Condition] Stack Recommendations
```

## Content Quality Assurance

### Checklist Per Page

**SEO Basics:**
- [ ] Target keyword in H1 (only once)
- [ ] Meta description (155-160 chars, includes target keyword)
- [ ] URL slug matches topic
- [ ] Proper heading hierarchy (H2-H4)
- [ ] Minimum 1,200 words
- [ ] Minimum 2 internal links

**Brand Voice:**
- [ ] Evidence-based (3+ sources minimum)
- [ ] Scientific but accessible language
- [ ] Appropriate disclaimers (when needed)
- [ ] Consistent tone with existing content
- [ ] No unsupported claims

**Internal Linking:**
- [ ] 3-5 internal links per page
- [ ] Links to related conditions
- [ ] Links to relevant herbs/compounds
- [ ] Links to related guides
- [ ] Links anchor text is natural/descriptive

**Content Depth:**
- [ ] Evidence for each recommendation
- [ ] Mechanisms explained
- [ ] Dosing guidance
- [ ] Safety considerations
- [ ] Contraindications noted
- [ ] FAQ answers specific claims

**Formatting:**
- [ ] Subheadings used effectively
- [ ] Tables/comparisons where relevant
- [ ] Lists are readable
- [ ] Images (if used) have alt text
- [ ] No broken internal links

## Contingency Planning

### If Rankings Don't Materialize

- A/B test titles and meta descriptions
- Test internal linking strategies
- Increase citation depth
- Add more unique data/tables
- Consider outreach for backlinks
- Evaluate keyword targeting (may be too competitive)

### If Content Production Falls Behind

- Reduce content depth (1,500 words vs. 2,000)
- Focus on highest-impact keyword clusters first
- Use AI content assistance for drafting
- Outsource to freelance writers
- Prioritize buyer intent pages (highest ROI)

### If Competing Sites Outrank You

- Strengthen evidence density (add more sources)
- Create unique data/comparisons
- Build topical authority faster (batch similar topics)
- Increase internal linking
- Consider guest posting for authority
